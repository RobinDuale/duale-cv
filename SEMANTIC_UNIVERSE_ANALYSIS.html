CONTENT WRITING GUIDELINES


Tone & Style Principles

  • **PE Audience:** Use investment thesis language, quantified outcomes, risk management vocabulary
  • **Professional:** Confident but not arrogant; demonstrated through results
  • **Technical Credibility:** Explain methodologies, frameworks, tools
  • **Global Professional:** English and French equally polished, cultural nuance respected

Semantic Writing Best Practices


1. Cluster-Based Keywords

Don't force keywords; rather, develop natural content around semantic clusters:

  • **Transformation cluster:** Methodology → Implementation → Results → Framework
  • **Value creation cluster:** Mechanism → Measurement → Communication → Proof
  • **AI cluster:** Current state → Opportunity → Approach → Implementation → ROI

2. Supporting Keyword Density

  • **Primary keyword:** 1-2% (natural occurrence in 300-400 word section)
  • **Secondary keywords:** 0.5-1% each
  • **Long-tail variations:** As natural narrative requires

3. Examples & Proof

Every semantic claim should have supporting proof:

  • "AI transformation": Link to specific implementation (predictive analytics, automation)
  • "Operational optimization": EBITDA improvement numbers (7%→23% TISSOT example)
  • "Transformation methodology": 5-phase framework with timeline
